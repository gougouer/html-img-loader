# html-img-loader

![](https://img.shields.io/npm/dw/img-html-loader)
![](https://img.shields.io/npm/l/img-html-loader)

## 引用
`npm i img-html-loader`

## 示例
```
{
        test:/\.(html|htm)$/i,
         use:'html-img-loader', // 解析 html中的图片资源
}
```
